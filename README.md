# Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites in Tableau

## Project Description
This project analyzes UNESCO World Heritage Sites using Python, Tableau, and Flask. It provides interactive dashboards and stories to visualize heritage distribution, endangered sites, and historical trends.
